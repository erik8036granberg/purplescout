Ambu

3D visualization

Purple Scout aided Ambu to create realtime 3D product visualization - a sales tool for the sales force. This interactive 3D visualization of Ambu products provided the opportunity to go into depth with unique components on different products.

Purple Scout created animated films that showed details on each product and integrated the Ambu sales database with product variants. This was all done as realtime 3D on iPads.
